The Benefits of Cloud Computing for Businesses

I have decided to do my 4th Year project on moving to a cloud-based infrastructure from an on-premise infrastructure. This project will discuss what is cloud computing, the different deployment models available (private, public, hybrid) and the cloud services available (IaaS, PaaS, SaaS). Once these have been discussed in depth, I will be talking about 2 of the leading market services available for cloud-based infrastructure (AWS and Azure). These two cloud services dominate the market making up over 51% of all total cloud spent in 2021.

Cloud Computing

Cloud computing is the delivery of computing resources such as storage, servers, networking, software, and tools over the internet. Hosting these resources on the cloud as opposed to an on-site infrastructure has many benefits for businesses’ which will be discussed shortly. These resources are added on a pay-as-you-go basis, which is beneficial to businesses as more resources can be added under heavy workloads, and the opposite for lighter workloads. A cloud migration is directly related to data and IT modernization for businesses. Not only is this topic related to my course, but it is also an important technology in today’s world of computers.

The main reason I choose to do this topic was because before I got into college, I had always been hearing about this magical ‘cloud’ and had no idea what it was. To me it seems that most businesses are moving to the cloud for the benefits it provides, such as reduced IT costs, scalability, reliability, flexibility, security, amongst many others. As this infrastructural move to the cloud is looking optimal for most businesses in the near future, I think as a student it would be quite beneficial to further my knowledge in this field because when I graduate my add-on year, I will be seeking employment in this sector.

Implementation

Once the theory for my project has concluded, this is mainly the previously talked about topics, I will be moving onto the implementation phase of the project. I am proposing that an IT manager within a business is recommending that they move some/all of the resources (storage, compute, network) to the cloud for the benefits it provides. I will be talking about the on-premise infrastructure before carrying out the implementation. The implementation will consist of a scaled down infrastructure of the previous mentioned resources on both Amazon Web Services and Microsoft Azure. Once I have implemented these resources to the cloud, I will talk about how it was done, and how to migrate the current physical on-site infrastructure to the cloud. This will contain things such as a script to migrate on-premise VM’s to a cloud-based service with a PowerShell script.

 

Conclusion

Once the implementations have been completed, I will be drawing up a conclusion on which of the two discussed services is better suited for a cloud-based migration. This will discuss the unique features to each, advantages, and disadvantages etc. This will also include what type of cloud deployment model will be used, public / private cloud, or the use of both in a hybrid cloud. The option selected will be compared with its competitor with things like cost to implement, deployment times, services available, disaster recovery, security etc. The selected cloud deployment method will more than likely be a hybrid infrastructure with some resources located on the business premises, with some extra resources on the cloud such as virtual machines, storage, and network resources.
